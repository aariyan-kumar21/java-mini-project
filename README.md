
# Java mini project

**1. Problem Statement**

To develop a simple Java program that allows users to search for available flight routes between two cities and display matching flights along with their ticket prices.

**2. Project Description**

This project is a Flight Route Search System implemented using Java. It uses a predefined dataset of flights stored in a 2D array. The program takes user input for departure and destination cities and searches through the dataset to find matching routes.

If matching flights are found, the program displays the airline name and ticket price. If no flights match the given route, it informs the user accordingly.

This project demonstrates basic programming concepts such as arrays, loops, and string handling in Java.

**3. Concepts Used** 

🔹 1. Arrays (2D Array)

Used to store multiple flight records in tabular form.

🔹 2. Loops (for loop)

Used to iterate through all flight data.

🔹 3. Conditional Statements (if)

Used to check if source and destination match.

🔹 4. String Methods

equalsIgnoreCase() for case-insensitive comparison.

🔹 5. Scanner Class

Used for taking input from the user.

🔹 6. Boolean Variable

Used to track whether any matching flight is found.
